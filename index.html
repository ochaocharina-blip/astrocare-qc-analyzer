📊 Import ke Google Sheets
Langkah-langkah import file .xlsx ke Google Sheets:
  1. Klik Export to Excel di sebelah kiri → file .xlsx terunduh
  2. Buka sheets.new di tab baru
  3. Menu File → Import → tab Upload
  4. Drag & drop file .xlsx atau klik Browse
  5. Pilih Replace spreadsheet → Import data ✓
✅ File .xlsx langsung terbaca tanpa konversi — kolom tidak berantakan
